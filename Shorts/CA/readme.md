## Setup
This is a simple script that compares two integer tuples and returns those not available/present on both lists.

 - Install libries listed on the requirements.txt before running the
   script. 
 - Check command-line / terminal output incase GUI fails to show
   list of absent persons.


 **Libraries**

| Library | Doc. |
| ------ | ------ |
|Easygui| https://pypi.org/project/easygui/ |
|Tabula | https://pypi.org/project/tabula/ |
|Pandas | https://pandas.pydata.org/ |
